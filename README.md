# 💰 FinTrack - Personal Finance Tracker

A modern fintech-style Personal Finance Tracker Web Application built using HTML, CSS, JavaScript, Chart.js, and LocalStorage.

The project features a premium dark mode dashboard UI with responsive layouts, expense management, analytics charts, budget tracking, animations, and data persistence.

---

# 🚀 Features

## ✅ Dashboard Features

* Modern Fintech SaaS UI
* Dark / Light Mode
* Glassmorphism Cards
* Sticky Navbar
* Sidebar Navigation
* Responsive Design

---

## ✅ Expense Management

* Add Expenses
* Edit Expenses
* Delete Expenses
* Search Expenses
* Filter by Category
* Sort by Amount & Date
* Real-time Updates

---

## ✅ Budget Management

* Set Monthly Budget
* Dynamic Remaining Balance
* Animated Progress Bar
* Budget Warning Alerts

---

## ✅ Analytics

* Pie Chart (Category Spending)
* Bar Chart (Expense Comparison)
* Live Chart Updates
